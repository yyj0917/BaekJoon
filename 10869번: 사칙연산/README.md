## 문제

[문제 링크](https://boj.kr/10869)

<p>두 자연수 A와 B가 주어진다. 이때, A+B, A-B, A*B, A/B(몫), A%B(나머지)를 출력하는 프로그램을 작성하시오.&nbsp;</p>

## 입력

<p>두 자연수 A와 B가 주어진다. (1 ≤ A,&nbsp;B ≤ 10,000)</p>

## 출력

<p>첫째 줄에 A+B, 둘째 줄에 A-B, 셋째 줄에 A*B, 넷째 줄에 A/B, 다섯째 줄에 A%B를 출력한다.</p>
