## 문제

[문제 링크](https://boj.kr/10951)

## 입력

<p>입력은 여러 개의 테스트 케이스로 이루어져 있다.</p>

<p>각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 &lt; A, B &lt; 10)</p>

## 출력

<p>각 테스트 케이스마다 A+B를 출력한다.</p>
